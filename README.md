# Blue Archive Sensei no Enjo

## Project Overview
This project is a fan-made tool for the **Blue Archive** community, designed to be lightweight, fast, and interactive.

## Tech Stack
- **Framework**: React (for high interactivity and ecosystem)
- **Build Tool**: Vite (for extremely fast builds and lightweight output)
- **Language**: TypeScript (for maintainability and community contribution)
- **Styling**: CSS Modules / Plain CSS (initially)

## Goals
- **Performance**: Minimize initial load times.
- **Interactivity**: Support complex interactions (Drag & Drop for team building, Resource Calculation).
- **Maintenance**: Clean codebase for community contributions.

## Setup Instructions
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
